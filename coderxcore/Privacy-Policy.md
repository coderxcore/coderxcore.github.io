# Privacy Policy / 隐私政策

**English**

This extension respects and protects user privacy. This Privacy Policy explains how the extension handles (or does not handle) user data.

1. Data Collection

   This extension **does not collect, upload, store, or share any user data**.
   It does not collect personal information, browsing history, user input, device information, location data, or any other identifiable data.

2. Data Usage

   All features of the extension run **entirely on the user’s local device**.
   Any data generated (such as local settings, cache, or temporary content) is stored only within the local browser environment and is never transmitted externally.

3. Data Transmission

   This extension **has no server-side components** and **does not communicate with any remote servers**.
   There is no data upload, synchronization, analytics, or remote processing of any kind.

4. Third-Party Services

   This extension **does not use any third-party analytics, advertising, tracking, or data collection services**.

5. Permissions

   Any browser permissions requested by the extension are used solely to provide its core functionality and are **not** used for tracking, monitoring, or data collection.

6. Changes to This Privacy Policy

   If future updates change how privacy-related matters are handled, this Privacy Policy will be updated accordingly.

7. Contact

   If you have any questions regarding this Privacy Policy, please contact the developer via the extension’s listing page.
   

**中文（简体）**

本插件高度重视用户隐私与数据安全。本隐私政策用于说明本插件如何处理（或不处理）用户数据。

1. 数据收集

   本插件**不会收集、上传、存储或共享任何用户数据**。
   不收集包括但不限于：个人身份信息、浏览记录、输入内容、设备信息、位置信息等。

2. 数据使用

   插件的所有功能均在**用户本地设备上运行**。
   所有数据（如本地配置、缓存或临时内容）仅存储在浏览器本地环境中，不会发送至任何服务器。

3. 数据传输

   本插件**不包含任何服务器端组件**，也**不会与任何远程服务器进行通信**。
   不存在数据上传、同步、分析或远程处理行为。

4. 第三方服务

   本插件**不使用任何第三方分析、统计、广告或追踪服务**。

5. 权限说明

   插件申请的浏览器权限仅用于实现对应功能，不用于用户追踪、行为分析或数据收集。

6. 隐私政策的变更

   如插件功能发生变化并涉及隐私处理方式调整，本隐私政策将相应更新。

7. 联系方式

   如对本隐私政策有任何疑问，可通过插件发布页面提供的联系方式进行反馈。
